# Integer

- `.digit` - Digits on a number
- `.abs` - Absolute value

# String

- `.split` - Split string into array
- `.chars` - Chars of string
- `.capitalize` - First letter high
- `.upcase` - All letters high
- `.downcase` - All letters low
- `.strip` - Remove white spaces

# Structures

```ruby
case year
when 0..2000
  'old'
when 2000..10000
  'new'
end
```
